# 01 CLI MVP：本地论文目录导入 + 问答 + 引用

## 任务状态规则

- 未完成：`- [ ]`
- 已完成：`- [x]`
- 已删除/取消：`- [ ] ~~任务内容~~`
- 每完成一个任务，就只勾选对应任务，不批量勾选。

## 目标

做出一个可运行的 CLI MVP：用户指定本地 PDF 目录，系统解析论文、建立本地索引，然后用户用自然语言提问，系统返回答案和引用来源。

- [x] 确认 Python 版本目标为 Python 3.11+
- [x] 创建基础 Python 项目结构
- [x] 创建 CLI 入口模块
- [x] 创建配置模块

## 2. 依赖选择

- [x] 选择 PDF 解析库，默认使用 PyMuPDF
- [x] 选择 token/chunk 工具，默认使用 tiktoken
- [x] 选择 LLM/Embedding 调用方式，默认兼容 OpenAI API
- [x] 选择本地向量存储，默认优先 Chroma 或 FAISS 中较轻量的一种
- [x] 选择元数据存储，MVP 默认使用 SQLite 或 JSONL
- [x] 固定最小依赖清单

## 3. 数据模型

- [x] 对损坏或无法解析 PDF 给出错误记录，不中断整个目录导入
